## venv in local
```
cd to-do-app/fastapi

[create venv]
python venv -m venv

[access venv]
venv\Scripts\Activates.ps1 (windows)
source venv/bin/activate (mac)

[select interpreter]
get-command python (windows)
which python (mac)

[install library]
pip install -r requirements.txt

[run app]
uvicron main:app --reload
```
